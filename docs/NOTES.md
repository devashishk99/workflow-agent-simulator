# Important Notes

## Database Migration Update

**Vercel Postgres is no longer available** (discontinued December 2024).

### Current Recommendation: Neon

- **Via Vercel Marketplace**: Easiest integration, automatic environment variables
- **Direct Setup**: Go to neon.tech, create database, add connection string manually

### Alternatives
- Railway (railway.app)
- Supabase (supabase.com)
